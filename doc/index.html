What is a Waveguide?

A Rectangular Waveguide is essentially a hollow metallic pipe used to carry high-frequency radio waves (microwaves). Unlike a standard wire which carries electrical current, a waveguide carries electromagnetic fields.

The "Tunnel" Analogy Imagine a tunnel. If you try to speak in a very low voice (low frequency), the sound dies out quickly. But if you yell (high frequency), the sound echoes and travels far. A waveguide acts like this tunnel: it acts as a High-Pass Filter, allowing only frequencies above a certain "Cutoff Frequency" to pass through.

Understanding Wave Modes

Inside the waveguide, the electromagnetic waves bounce off the walls in specific patterns called "Modes." The WAVELINK simulator visualizes these three distinct types of modes:

1. Transverse Electric (TE) Mode

In this mode, the Electric Field (E) is entirely "transverse," meaning it is perpendicular to the direction the wave is moving (Z-axis). It never points forward or backward.

  • Key Feature: There is no Electric field in the direction of propagation (\( E_z = 0 \)).
  • Magnetic Field: Has a component that points in the direction of travel (\( H_z \neq 0 \)).
  • Dominant Mode: The TE10 mode is the most important one. It requires the smallest waveguide dimensions to propagate.

2. Transverse Magnetic (TM) Mode

In this mode, the Magnetic Field (H) is entirely transverse. It stays perpendicular to the direction of travel.

  • Key Feature: There is no Magnetic field in the direction of propagation (\( H_z = 0 \)).
  • Electric Field: Has a component that points in the direction of travel (\( E_z \neq 0 \)).
  • Constraint: TM modes cannot exist if either mode index \( m \) or \( n \) is zero. The lowest TM mode is TM11.

3. Transverse Electromagnetic (TEM) Mode

Why doesn't TEM work in Rectangular Waveguides?
The WAVELINK simulator will disable the "Start" button if you select TEM mode. This is correct physics!

In a TEM mode, both the Electric and Magnetic fields are perpendicular to the direction of travel. A TEM wave requires two separate conductors (like a Coaxial cable) to support a voltage difference. A rectangular waveguide has only one conductor (the walls), so it cannot support TEM waves.

Surface Current Density (\( \vec{J}_s \))

The WAVELINK simulator includes a 3D view of the Surface Current. These are real electrical currents flowing on the inner walls of the copper waveguide.

The Formula

The Surface Current Density \( \vec{J}_s \) is calculated directly from the Magnetic Field \( \vec{H} \) at the boundary:

$$ \vec{J}_s = \hat{n} \times \vec{H} $$

Where \( \hat{n} \) is the unit vector normal (perpendicular) to the surface pointing inward. This explains why the arrows in the 3D view flow perpendicular to the magnetic field lines near the walls.

Code & Visualization Logic

How does MATLAB create these 3D graphics from simple math formulas? This section explains the key functions used in the WAVELINK code.

1. Creating the Wave Animation

The simulation doesn't "move" the wave. Instead, it recalculates the wave's phase at a slightly different position in every frame. This creates the illusion of movement (just like a flipbook). 

% Animation Logic (inside updateSimulationStep)
app.zPos = app.zPos - zSampDist;  % Shift Z position slightly
z = app.zPos;                     % Update Z variable

% Recalculate Fields with new Z
[Ex, Ey, Ez] = fieldsTEmn(m, n, a, b, X, Y, Z, ...);

2. Visualizing Electric Fields (Surface Plot)

To show the colorful Electric Field, the code uses the surf command. This creates a "Texture Map" where the color represents the field strength (Red = Positive Max, Blue = Negative Max). 

% Drawing the colored surface
surf(app.UIAxes, xImage, yImage, zImage, ...
     'CData', real(Exz1), ...       % Color Data = E-Field Strength
     'FaceColor', 'texturemap');    % Smooth interpolation

3. Visualizing Magnetic Fields (Vector Plot)

Magnetic fields have direction, so we cannot just use colors. We use the quiver3 command, which draws thousands of tiny arrows. The length and direction of each arrow show the field's behavior at that point. 

% Drawing the vector arrows
quiver3(app.UIAxes_2, X, Y, Z, ...
        real(Hx), real(Hy), real(Hz)); % Arrow components (u,v,w)

4. Calculating Field Components

The math is solved in separate functions (fieldsTEmn.m and fieldsTMmn.m) to keep the code clean. Here is how the TE mode Magnetic Field \( H_z \) is translated into code:

$$ H_z = C \cos\left(\frac{m\pi x}{a}\right) \cos\left(\frac{n\pi y}{b}\right) e^{-j\beta z} $$ 
% MATLAB Implementation
kx = m * pi / a;
ky = n * pi / b;

% ".*" allows calculating thousands of points at once (Vectorization)
Hz = C .* cos(kx .* X) .* cos(ky .* Y) .* exp(-1j .* beta .* Z);

Glossary of Parameters

Before using the simulator, familiarize yourself with these inputs found on the control panel:

Parameter Variable Description
Width \( a \) The wider dimension of the waveguide cross-section. Usually determines the cutoff frequency.
Height \( b \) The narrower dimension. Usually \( b = a/2 \) in standard waveguides.
Mode Indices \( m, n \) Integers (0, 1, 2...) representing the number of "half-wave bumps" across the width (\(m\)) and height (\(n\)).
Frequency \( f \) The operating frequency of the source. Must be higher than \( f_c \) to propagate.
Permittivity \( \epsilon_r \) Dielectric constant of the material inside. Air = 1. Higher values slow the wave down.

Governing Physics Formulas

The simulator uses the exact solutions to Maxwell's Equations for a rectangular boundary.

1. Cutoff Frequency (\(f_c\))

The frequency threshold below which the wave cannot travel.

$$ f_c = \frac{1}{2\sqrt{\mu\epsilon}} \sqrt{ \left(\frac{m}{a}\right)^2 + \left(\frac{n}{b}\right)^2 } $$

2. TE Mode Fields

Derived from the longitudinal magnetic field \(H_z\):

$$ H_z = C \cos\left(\frac{m\pi x}{a}\right) \cos\left(\frac{n\pi y}{b}\right) e^{-j\beta z} $$ $$ E_x = \frac{j\omega\mu}{h^2} \frac{\partial H_z}{\partial y} \quad , \quad E_y = -\frac{j\omega\mu}{h^2} \frac{\partial H_z}{\partial x} $$

3. TM Mode Fields

Derived from the longitudinal electric field \(E_z\):

$$ E_z = C \sin\left(\frac{m\pi x}{a}\right) \sin\left(\frac{n\pi y}{b}\right) e^{-j\beta z} $$ $$ H_x = \frac{j\omega\epsilon}{h^2} \frac{\partial E_z}{\partial y} \quad , \quad H_y = -\frac{j\omega\epsilon}{h^2} \frac{\partial E_z}{\partial x} $$
